Stack Fun (Linked List Version)

Instructions
----------------------------------
To run this program efficiently, do the following steps
Unzip the file 
Navigate to the directory where the unzipped files are
Type "gcc -o Floodfill main.c"
Press Enter
Type "./Floodfill (the name of the picture).txt"
Press Enter
Follow the instructions below
1. Type in the commands to run and choose which picture to use
2. Type in the number of the row you would like to change
3. Type in the number of the column you would like to change
4. Pick a desired color
5. Press enter and enjoy the result
